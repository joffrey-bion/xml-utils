Provides some helper classes to handle XML manipulation and serialization easily.

## Usage

### Add the dependency in Gradle

    repositories {
        maven {
            url  "http://dl.bintray.com/joffrey-bion/maven"
        }
    }

    dependencies {
        compile 'org.hildan.utils:xml-utils:0.9'
    }
